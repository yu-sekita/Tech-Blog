===========
Tech Blog
===========


目的
=====
簡単なBlogを作りながらDjangoを学ぶこと。


ツールのバージョン
====================
:Python:     3.7.3
:pip:        19.1.1


インストールと起動方法
========================

    $ git clone https://github.com/yu-sekita/Tech-Blog
    $ cd Tech-Blog
    $ pipenv install
    $ pipenv shell
    $ cd Tech_Blog
    $ ./manage.py runserver
